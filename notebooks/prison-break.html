Analysing data¶

Prison Helicopter Escapes¶

I chose this project because it is set as part of the Dataquest course optional projects. It's interesting because it:

  1. provides an opportunity to practice skills learnt to date
  2. provides an opportunity to work with Jupyter notebook
  3. seeks to analyse an unusual dataset which I didn't even know existed!

It's interesting also because the analysis might provide information for law enforcement agencies to prevent escapes of this kind.

Conclusion¶

The project gave me skills in using Jupyter and in developing code to analyse the dataset. Results showed fascinating data indicating France has a significant and decades-long problem with escapes via helicopter. Further analysis might provide more detail about how France and other countries might deal with this issue.
